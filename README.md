# paypal-provider-service

Microservice which help to make api calls (createOrder and captureOrder) to the paypal system
The project builds an online store (merchant) that uses PayPal as its Payment Service Provider (PSP). A merchant is the business (e.g. our T‑shirt store) that sells goods, while the PSP (PayPal) handles payment processing. In other words, PayPal is a third-party gateway that manages the entire transaction – from collecting card or wallet details to communicating with banks and depositing funds into the merchant’s account. This separation ensures the merchant never handles raw payment data directly.
